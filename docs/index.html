ECGDT: ECG Diagnosis Tool.

System for aided-detection of cardiac conditions in ECG recordings

Download

What is ECGDT?

ECGDT is a web app which helps clinicians with the diagnosis of six main conditions into ECG (Electrocardiogram) digitized records.

The system performs basic ECG detection, delineation and diagnosis tasks, using ECG records in ASCII or WFDB format.

Free to Use

Free to download and use for any purpose

Open Source

Modify and improve it as you wish

Single-channel detection

Detection of heart beats in individual channels

Global beats detection

Detection of multi-channel heart beats combining information from all channels available

Beat wave delineation

Identification of waves and segments present inside a given beat

Computer-aided diagnosis

Capable of assessing clinicians with the detection of bradycardia, ischemia, myocardial infarction, tachycardia, ventricular hypertrophy and Wolff-Parkinson-White syndrome

Extra features

Graphic representation, record exportation, basal beat creation,...

Easy to use

Move over the menus in a very simple way
